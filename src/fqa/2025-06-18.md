# 股票分割的紀錄方式

以 2025-06-18 0050 「1拆4」為例

假設 (原本持有) 1張 ➡️ (分割後) 4張
可以這樣紀錄

|     日期     | 買/賣 |  代號  | 買入股數 | 買入價格 | 賣出股數 | 賣出價格 |
|:----------:|:---:|:----:|:----:|:----:|:----:|:----:|
| 2025-06-18 | 賣出  | 0050 |      |      | 1000 |      |
| 2025-06-18 | 買進  | 0050 | 4000 |      |      |      |

上表空白處不用填
最後再把自動輸入區的「折讓後手續費」改成 0 (分割沒有成本)
