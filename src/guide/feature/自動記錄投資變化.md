#### 試算表將自動將以「週」為單位下數據記錄下來，方便觀察投資走勢的變化
 
- 投入成本
- 市值
- 現金
- 已實現損益
- 未實現損益

![](/images/台股訂閱版/儀表板_自動記錄.gif)

::: tip 操作小技巧
- 調整想要顯示的走勢圖的時間區間 → 走勢圖下方的時間軸 or 左上角的 zoom
- 圖中顯示的項目可以自行選擇　　 → 點選表格標題的下拉選單
:::

::: note 目前試算表的手機 APP 尚不支援時間線圖
:::
